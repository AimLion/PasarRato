# PasarRato
Juego en la consola de python para "pasar el rato"

## Detalles
Juego autónomo para observar si ganas o no el combate.

Ahora el usuario puede elegir entre:
- Iniciar el combate
- Confirmar el combate
- Repetir el combate
- Escapar del combate
